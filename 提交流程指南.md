## 🔧 第二步：上传代码

### 1. 打开支付宝开发者工具

### 2. 修改项目配置
打开 `project.config.json`，修改 appid：
```json
{
  "appid": "你的真实 AppID"
}
```

### 3. 在开发者工具中上传
- 点击右上角「上传」按钮
- 填写版本号：`1.0.0`
- 填写版本说明：
```
首次发布版本
- 提供8大类恋爱话术
- 提供6大类恋爱支招
- 支持收藏功能
- 支持一键复制
- 支持难度筛选
```
- 点击「确定」上传

## 🌐 第三步：登录开放平台

### 1. 访问支付宝开放平台
地址：https://open.alipay.com

### 2. 登录账号
使用您的支付宝账号登录

### 3. 进入小程序管理
- 点击「控制台」
- 选择「小程序」
- 进入您的小程序

---

## 📝 第四步：填写基本信息

### 1. 小程序信息设置

#### 基本信息
- **小程序名称**：恋爱话术助手
- **小程序简介**：复制《小程序简介.txt》中的「短版简介」
- **小程序描述**：复制《小程序简介.txt》中的「标准版简介」
- **小程序 LOGO**：上传准备好的 LOGO 图片

#### 类目选择
- 一级类目：生活服务
- 二级类目：生活助手

#### 服务区域
- 选择：中国大陆

### 2. 联系方式
- **客服邮箱**：填写您的邮箱（必填）
- **客服电话**：可选填
- **客服时间**：工作日 9:00-18:00（建议）

---

## 📄 第五步：配置隐私和协议

### 1. 隐私设置
- 点击「隐私设置」
- 数据收集说明：
  - 勾选「使用设备信息」（用于优化体验）
  - **不勾选**其他项（本小程序不收集其他信息）
- 隐私政策链接：
  - 如有自己的网站，可上传隐私政策并填写链接
  - 如无网站，直接在「隐私政策」文本框中粘贴《隐私政策.txt》内容

### 2. 用户协议
- 点击「用户协议设置」
- 复制《用户服务协议.txt》内容
- 粘贴到「用户服务协议」文本框中

---

## 🎨 第六步：配置小程序展示

### 1. 上传截图
点击「版本管理」→「开发版本」→「体验版设置」

上传5张产品截图（按顺序）：
1. 首页截图 - 展示四个主要功能入口
2. 话术列表页 - 展示话术分类和列表
3. 话术详情页 - 展示话术详细内容
4. 恋爱支招列表 - 展示支招分类和文章
5. 收藏页面 - 展示收藏管理功能

**截图拍摄方法：**
- 使用支付宝开发者工具的「真机预览」功能
- 在真机上截图
- 使用图片编辑工具调整为 750*1334 尺寸

### 2. 设置搜索关键词
建议关键词（最多10个）：
- 恋爱
- 话术
- 聊天
- 表白
- 约会
- 沟通技巧

## 🎯 第七步：版本提交

### 1. 检查开发版本
- 进入「版本管理」
- 查看「开发版本」
- 确认版本号为 `1.0.0`
- 确认代码已上传

### 2. 提交审核
- 点击「提交审核」按钮
- 再次确认以下信息：
  - [ ] 版本号正确
  - [ ] 版本说明完整
  - [ ] 截图已上传
  - [ ] 基本信息已填写
  - [ ] 隐私政策已配置
  - [ ] 用户协议已配置
  - [ ] 类目选择正确

### 3. 填写测试信息
- **是否需要测试账号**：选择「否」（本小程序无需登录）
- **测试说明**：
```
本小程序无需注册登录，可直接使用所有功能。
测试路径：
1. 进入首页，点击「恋爱话术」
2. 选择分类查看话术列表
3. 点击话术卡片查看详情
4. 测试复制和收藏功能
5. 返回首页，点击「恋爱支招」测试文章功能
6. 点击「我的收藏」查看收藏管理
```

### 4. 确认并提交
- 仔细检查所有信息
- 勾选「我已阅读并同意相关协议」
- 点击「确认提交」

---

## ⏳ 第八步：等待审核

### 审核时长
- 通常：1-3个工作日
- 快速通过：24小时内
- 如有问题：可能需要修改后重新提交

### 审核状态查看
- 在「版本管理」中查看审核状态
- 状态说明：
  - 「审核中」：等待审核
  - 「审核成功」：可以上架
  - 「审核失败」：需要查看原因并修改

### 审核失败处理
如果审核失败：
1. 查看审核意见
2. 根据意见修改代码或资料
3. 重新上传并提交

---

## 🚀 第九步：小程序上架

### 审核通过后
1. 进入「版本管理」
2. 在「审核版本」中看到审核通过的版本
3. 点击「上架」按钮
4. 确认上架信息
5. 点击「确定上架」

### 上架成功
- 上架成功后，用户可以通过支付宝搜索找到你的小程序
- 可以在「版本管理」→「线上版本」中看到已上架的版本

## ⚠️ 注意事项

### 审核要点
1. **内容合规**
   - 确保所有话术和文章内容健康、积极
   - 不涉及色情、暴力、违法内容
   - 不涉及敏感话题

2. **功能完整**
   - 所有页面可正常访问
   - 所有按钮可正常点击
   - 无明显 Bug 和报错

3. **信息真实**
   - 填写的联系方式真实有效
   - 功能描述准确无误
   - 不夸大宣传

### 常见问题

**Q1：审核被拒绝说内容不符合规范？**
A：检查话术内容，删除可能引起误会的内容，确保所有内容积极健康。

**Q2：审核被拒绝说功能描述不清？**
A：完善功能描述，详细说明小程序用途和使用场景。

**Q3：如何修改已上架的小程序？**
A：上传新版本，填写更新说明，重新提交审核，审核通过后上架即可覆盖旧版本。

**Q4：可以更改小程序名称吗？**
A：可以，但一年只能更改3次，谨慎修改。
